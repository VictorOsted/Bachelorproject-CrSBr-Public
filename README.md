The repository is seperated into three parts.

## FM Heisenberg chain
This folder contains the Julia code for simulating a ferromagnetic Heisenberg chain. The Python code is for visualising the dispersion wave intensities.

## AFM Heisenberg chain
The same as for FM Heisenberg chain instead of it now being an antiferromagnetic Heisenberg chain.

## CrSBr
This folder contains both SQUID data for CrSBr and the .cif file for CrSBr. The Julia code simulates CrSBr with all relevant exchanges. It also simulates polarized neutron scattering to determine the antisymmetric exchange (DM-exchange). The data used for SQUID data is unpublished and therefore has been omitted.
